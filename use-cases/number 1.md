# USE CASE: 1 - All the countries in the world organised by largest population to smallest.

## CHARACTERISTIC INFORMATION

### Goal in Context

As a system user I want to produce a report of all the countries in the world organised from largest population to smallest population so that I can see which countries have the largest and smallest population in the world.
